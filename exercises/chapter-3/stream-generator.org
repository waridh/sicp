** Ones
Streams that just generates infinite ones

#+begin_src racket :exports code
(define ones (cons-stream 1 ones))
#+end_src
** Integers
Streams that generates incrementing integers.

#+begin_src racket :exports code
(define integers (cons-stream 1 (add-streams ones integers)))
#+end_src
